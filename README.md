# Habitus

Habitus is an open-source iOS + macOS interval reminder app built with Swift 6, SwiftUI, and SwiftData.

It fills a gap in the Apple ecosystem: recurring intra-day reminders with flexible scheduling, iCloud sync, and a native macOS menu bar - without a backend or subscription.

## What Habitus does

- Set reminders on a repeating interval within part of a day
- Schedule reminders at specific times
- Choose the days each reminder applies to
- Snooze or skip reminders in the moment
- Track completions over time
- Sync seamlessly between iPhone and Mac with iCloud

## Why it exists

iOS lets you set a reminder at a fixed time, but it will not remind you to do 10 pushups every 30 minutes between 10am and 6pm. Habitus is built for that kind of routine.

It starts as a personal tool for recurring habits and interval-based prompts, then grows into an open app others can use, inspect, and improve.

## Platform focus

- iOS app for creating, receiving, and acting on reminders throughout the day
- macOS companion with native desktop support and menu bar access
- Shared local-first data model powered by SwiftData and iCloud sync

## Open source

Habitus is built in the open. If it solves a problem for you too, contributions are welcome.
